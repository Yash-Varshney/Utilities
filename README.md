# Utilities
A collection of python scripts that may come in handy in everyday life at IIT Mandi

# CGPA.py
Prints SGPA for each semester and total CGPA obtained
## Requirements
```
pip3 install requests
pip3 install bs4
pip3 install lxml
pip3 install pandas
pip3 install getpass
pip3 install threading
pip3 install sys
pip3 install time
```
## Run ( In Terminal )
```
python3 CGPA.py
```
